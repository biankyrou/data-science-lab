## Indexação com NumPy

A indexação em NumPy permite acessar e modificar elementos ou subconjuntos de arrays de forma eficiente.

### Acessando Elementos

Para acessar elementos individuais em arrays 1D:

```python
a = np.array([10, 20, 30, 40])
print(a[2])  # Resultado: 30
```
